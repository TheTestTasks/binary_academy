Реализовать REST API из домашнего по HTTP с помощью http модуля node.js. Сделать отдельную программу, выполняющую то же самое с помощью expressjs. Данные сгенерировать и хранить любым образом

```
  ┌─────────────────────────┬──────────────┬───────────────────────────────────────────────┐

  │                         │              │User-Agent: Internet-Explorer 9.0              │
  ├─────────────────────────┼──────────────┼───────────────────────────────────────────────┤

  │стране                   │     GET      │Host: 127.0.0.1:8000                           │
  │                         │              │User-Agent: Internet-Explorer 9.0              │

  │                         │              │Host: 127.0.0.1:8000                           │
```
